# Project name:

## Node monitor:
Node monitor is a monitoring app to check up/down time and notify user.

## git clone:

repos [git](https://github.com/bpinazmul18/node-monitor).

```bash
git clone https://github.com/bpinazmul18/node-monitor.git
```

## Installation:

```nodejs
yarn install
```

## Usage:

```nodejs
yarn start
```
